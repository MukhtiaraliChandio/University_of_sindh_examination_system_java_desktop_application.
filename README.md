# University_of_sindh_examination_system_java_desktop_application.
The University of Sindh Examination System is a Java-based desktop application designed to manage and automate examination-related activities for students, and administrators within the university.It provides a centralized platform to handle student records, course enrollment, examination scheduling,
The main goal of this system is to replace manual exam record keeping with a computerized, secure, and efficient solution, reducing errors, improving accessibility, and saving administrative time.
